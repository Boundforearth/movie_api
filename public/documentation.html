myFlix

A server side API that will give users access to infromation on movies, directors, etc.

Description Endpoint Request Type Request Body Response Body
Display the home page / GET none
Display a list of all movies /movies GET none JSON Object holding data about all the movies
Display information about the selected movie /movies/[MOVIE] GET None JSON Object containing information about a movie
Register a user into the system /users POST JSON Object holding user registration information Text message confirming that the user has successfully registered
Allow user to update their username /users/[USERNAME] PUT none Text message telling user that their username has been updated
Display a users list of favorite movies /users/[USERNAME]/mylist GET none JSON Object containing a list of the user's favortie movies
Add a movie to the user's list /users/[USERNAME]/mylist/[MOVIE] POST JSON Object containing the movie the user would like to add Text message letting the user know the movie was successfully added
Remove a movie from the user's list /users/[USERNAME]/mylist/[MOVIE] DELETE none Text message letting user know the movie was removed
Deletes a user from the system /users/[USERNAME] DELETE none Lets user know they successfully deleted their profile
Display information about a movie genre /genres/[GENRE] GET none JSON Object containing information about the selected genre
Display information about selected director /directors/[DIRECTOR] GET none JSON Object containing information about the selected director
