myFlix

A server side API that will give users access to infromation on movies, directors, etc.

Endpoint Description Request Type Request Body Response Body
/ Display the home page GET none
/movies Display a list of all movies GET none JSON Object holding data about all the movies
/movies/[MOVIE] Display information about the selected movie GET None JSON Object containing information about a movie
/users Register a user into the system POST JSON Object holding user registration information Text message confirming that the user has successfully registered
/users/[USERNAME] Allow user to update their username PUT none Text message telling user that their username has been updated
/users/[USERNAME]/mylist Display a users list of favorite movies GET none JSON Object containing a list of the user's favortie movies
/users/[USERNAME]/mylist/[MOVIE] Add a movie to the user's list POST JSON Object containing the movie the user would like to add Text message letting the user know the movie was successfully added
/users/[USERNAME]/mylist/[MOVIE] Remove a movie from the user's list DELETE none Text message letting user know the movie was removed
/users/[USERNAME] Deletes a user from the system DELETE none Lets user know they successfully deleted their profile
/genres/[GENRE] Display information about a movie genre GET none JSON Object containing information about the selected genre
/directors/[DIRECTOR] Display information about selected director GET none JSON Object containing information about the selected director
