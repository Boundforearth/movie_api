myFlix

A server side API that will give users access to infromation on movies, directors, etc.

Features

  • Allow People to register and login to their accounts
  • Allow Users add and remove movies from a list of their favorites
  • Users can update their profile information
  • Allow Users to browse movies and get information about a single movie
  • Users will be able to pull up more detailed information in directors
  • Users will be able to grab a description of a genre
  • Allow Users to delete their accounts

Endpoints and Methods

Endpoint Description Request Type Request Body Response Body
/ Display the home page GET none Text Message welcoming user to the app
/movies Display a list of all movies GET none JSON Object holding data about all the movies
/movies/[MOVIE] Display information about the selected movie GET None JSON Object containing information about a movie
/users Register a user into the system POST JSON Object holding user registration information. Username, Password, and Email are required.
Example:
"Username": "yourusername",
"Password": "yourpassword",
"Email": "youremail",
"Birthday": "yyyy-mm-dd"
JSON object of User's information. Will be in the same format as above, but an ID will be returned as well
/users/[USERNAME] Allow user to update their username PUT JSON object with the information to be updates
Example:
"Username": "newusername",
"Password": "yourpassword",
"Email": "youremail",
"Birthday": "yyyy-mm-dd"
Text message telling user that their username has been updated
/users/[USERNAME]/mylist Display a user's list of favorite movies GET none JSON Object containing a list of the user's favortie movies
/users/[USERNAME]/mylist/[MOVIE] Add a movie to the user's list POST none, as the movie ID is a paramater in the header Text message letting the user know the movie was successfully added
/users/[USERNAME]/mylist/[MOVIE] Remove a movie from the user's list DELETE none Text message letting user know the movie was removed
/users/[USERNAME] Deletes a user from the system DELETE none Lets user know they successfully deleted their profile
/genres/[GENRE] Display information about a movie genre GET none JSON Object containing the name and description for the desired genre
/directors/[DIRECTOR] Display information about selected director GET none JSON Object containing relevant information about the requested director
